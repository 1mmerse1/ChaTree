# ChaTree — AI 智能对话管理系统

基于 PySide6 的桌面端 AI 对话应用，支持 OpenAI 兼容的多种 LLM 后端，提供文件夹管理、消息间链接、追问批注等结构化对话管理能力。

## 功能概览

- **多模型支持** — 内置 17 个模型预设（OpenAI GPT-5/4 系列、Gemini、DeepSeek、通义千问、Moonshot），支持自定义 Base URL
- **文件夹管理** — 树形文件夹组织对话，拖拽排序
- **流式输出** — 实时 token 级流式渲染，支持 Markdown + KaTeX 数学公式排版
- **追问/批注** — 选中助手回复中任意文字 → 右键"追问"，右侧面板显示 AI 回答
- **消息链接** — 在不同对话的消息之间建立引用链接（"链接到…"），自动显示正向/反向链接
- **智能标签** — 每轮对话自动生成 3–5 个标签，支持手动增删
- **重试编辑** — 编辑上文消息后重新生成助手回复
- **深色主题** — 全局暗色配色

## 环境要求

- Python 3.8+
- pip

## 安装

```bash
# 克隆仓库
git clone https://github.com/1mmerse1/ChaTree.git
cd ChaTree

# 安装依赖
pip install PySide6 openai markdown
```

> **说明**：KaTeX 数学渲染所需的 JS/CSS 已随项目打包，无需额外安装。

## 运行

```bash
python main.py
# 或
python -m chaTree
```

## 使用指南

### 1. 配置 API

首次启动后，点击左下角 **⚙ 设置**：

1. 在模型列表中选择一个预设（如 `OpenAI · GPT-4o mini`）
2. 填入对应厂商的 **API Key**
3. 如需使用代理或第三方 API，可修改 Base URL
4. 点击保存

### 2. 开始对话

- 左侧 **+ 新建对话**，输入标题
- 在底部输入框输入消息，**Ctrl+Enter** 发送
- 助手回复支持 Markdown 渲染（表格、代码块、数学公式等）

### 3. 数学公式

支持LaTeX 公式的显示，由 KaTeX 引擎实时渲染

### 4. 追问（批注）

- 在助手回复中**选中一段文字**
- 右键 → **💬 追问「...」**
- 在弹出的对话框中输入问题
- 回答显示在右侧批注面板

### 5. 消息链接

在不同对话的消息之间建立引用关系：

- 选中文字 → 右键 → **🔗 链接「...」到...**
- 在弹出的对话框中浏览并选择目标对话/消息
- 消息下方显示链接徽章，点击跳转到目标
- 被链接的对话显示反向链接

### 6. 标签管理

- 每轮问答由AI自动生成标签，显示在消息和对话顶部
- 消息标签：点击 **+** 添加
- 对话标签：点击对话顶部的 **+** 按钮或右键标签进行管理

### 7. 其他操作

| 操作 | 方式 |
|---|---|
| 重命名对话 | 双击对话标题 |
| 重试回复 | 点击助手消息下方的"↻ 重试" |
| 查看反向链接 | 点击消息下方的"← 反向链接" |
| 删除对话 | 右键侧边栏对话项 |
| 新建文件夹 | 右键侧边栏空白处 |
