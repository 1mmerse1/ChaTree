# ChainTree — AI 智能对话管理系统

基于 PySide6 的桌面端 AI 对话应用，支持 OpenAI 兼容的多种 LLM 后端，提供文件夹管理、消息间链接、追问批注、支线展开、知识图谱等结构化对话管理能力。

## 功能概览

- **多模型支持** — 内置 17 个模型预设（OpenAI GPT-5/4 系列、Gemini、DeepSeek、通义千问、Moonshot），支持自定义 Base URL
- **文件夹管理** — 树形文件夹组织对话，拖拽排序
- **流式输出** — 实时 token 级流式渲染，支持 Markdown + KaTeX 数学公式排版
- **追问/批注** — 选中助手回复中任意文字 → 右键"追问"，右侧面板显示 AI 回答
- **支线对话** — 将批注扩展为多轮子对话（无严格 token 限制），右侧面板独立展示，支持标签和链接，知识图谱中以菱形节点呈现
- **消息链接** — 在不同对话的消息之间建立引用链接（"链接到…"），自动显示正向/反向链接
- **AI 自动推荐链接** — 每次 AI 回答后自动分析是否与历史轮次存在强关联，弹窗推荐至多 3 条链接供确认（可在设置中开关）
- **知识图谱** — pyvis + QWebEngineView 可视化对话结构，主线 + 支线节点 + 链接边，点击节点跳转
- **活跃热力图** — 按日期统计对话/支线活动量，颜色深浅展示活跃程度
- **智能标签** — 每轮对话自动生成 3–5 个标签，支持手动增删
- **多样化输入** — 输入框左侧 `+` 按钮上传代码/PDF/Word 文件，内容自动拼入消息上下文
- **重试编辑** — 编辑上文消息后重新生成助手回复
- **深色主题** — 全局暗色配色

## 环境要求

- Python 3.8+
- pip

## 安装

```bash
# 克隆仓库
git clone https://github.com/1mmerse1/ChaTree.git
cd ChaTree

# 安装依赖
pip install PySide6 openai markdown pyvis PyQtWebEngine PyPDF2 python-docx
```

> **说明**：KaTeX 数学渲染所需的 JS/CSS 已随项目打包，无需额外安装。

## 运行

```bash
python main.py
# 或
python -m chaTree
```

## 使用指南

### 1. 配置 API

首次启动后，点击左下角 **⚙ 设置**：

1. 在模型列表中选择一个预设（如 `OpenAI · GPT-4o mini`）
2. 填入对应厂商的 **API Key**
3. 如需使用代理或第三方 API，可修改 Base URL
4. 可选：勾选 "AI 自动推荐链接" 启用智能链接建议
5. 点击保存

### 2. 开始对话

- 左侧 **+ 新建对话**，输入标题
- 在底部输入框输入消息，**Ctrl+Enter** 发送
- 助手回复支持 Markdown 渲染（表格、代码块、数学公式等）

### 3. 上传文件

- 输入框左侧点击 **+** 按钮
- 支持格式：代码文件（`.py` `.js` `.java` `.go` 等）、PDF（`.pdf`）、Word（`.docx`）、纯文本（`.txt` `.md` 等）
- 选中文件后以标签形式显示在输入栏上方
- 点击标签 `✕` 可移除文件
- 发送时文件内容自动附在消息末尾（代码有语法高亮标记）
- PDF/DOCX 需安装对应库：`pip install PyPDF2 python-docx`（未安装时给出友好提示）

### 4. 数学公式

支持 LaTeX 公式的显示，由 KaTeX 引擎实时渲染。

### 5. 追问（批注）

- 在助手回复中**选中一段文字**
- 右键 → **💬 追问「...」**
- 在弹出的对话框中输入问题
- 回答显示在右侧批注面板

### 6. 支线对话

将批注扩展为多轮子对话，适合深入探讨某个特定话题：

- 在批注面板中点击 **📂 扩展为支线**
- 支线在右侧面板独立展示，支持多轮连续对话（无严格 token 限制）
- 支线内可手动添加标签、建立链接
- 支线在知识图谱中以**青色菱形节点**呈现，连接到主线的对应轮次
- 支线内不能再嵌套支线或批注
- 支线消息计入活跃热力图统计

### 7. 消息链接

在不同对话的消息之间建立引用关系：

- 选中文字 → 右键 → **🔗 链接「...」到...**
- 在弹出的对话框中浏览并选择目标对话/消息
- 消息下方显示链接徽章，点击跳转到目标
- 被链接的对话显示反向链接

**AI 自动推荐链接：**

- 在设置中开启 "AI 自动推荐链接"
- 每次 AI 回答完成后，自动分析当前轮次与历史轮次（同对话相隔 ≥5 轮，或跨对话）的关联
- 发现强关联时弹出确认窗口（至多 3 条），默认全选
- 点击 "创建选中链接" 或 "跳过"
- 关联标准严格：需具体概念/技术直接重合，而非仅同属大类（如"都是数学"不算）

### 8. 知识图谱

- 工具栏点击 **知识图谱** 按钮打开
- 主线轮次以**圆形节点**呈现，支线以**青色菱形节点**呈现
- 蓝色边：主线轮次顺序连接
- 橙色边：消息链接关系
- 青色边：支线连接到主线对应轮次
- 点击节点查看详情并跳转

### 9. 活跃热力图

- 工具栏点击 **热力图** 按钮打开
- 按日期汇总对话消息数和支线活动
- 颜色越深表示当天活动越多

### 10. 标签管理

- 每轮问答由 AI 自动生成标签，显示在消息和对话顶部
- 消息标签：点击 **+** 添加
- 对话标签：点击对话顶部的 **+** 按钮或右键标签进行管理

### 11. 其他操作

| 操作 | 方式 |
|---|---|
| 重命名对话 | 双击对话标题 |
| 重试回复 | 点击助手消息下方的 "↻ 重试" |
| 查看反向链接 | 点击消息下方的 "← 反向链接" |
| 删除对话 | 右键侧边栏对话项 |
| 新建文件夹 | 右键侧边栏空白处 |
